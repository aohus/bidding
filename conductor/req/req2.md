# track1: main page

## main page default

- 랜딩하면 기본 데이터 로딩되도록, 기본은 '개찰일시'
- 마지막 검색한 데이터 캐싱해서 새로고침해도 그대로 데이터 존재하도록 수정

## main page 시작/종료일

- 시작일/종료일 component를 한개의 달력에서 선택할 수 있게 수정

## main page 검색 조건

- 참가제한지역명 (선택)에 "경기, 여주" 이렇게 여러개 입력하면 api 2번 호출해서 모든 데이터 반환
- 업종명 (선택)에 "조경식재,나무병원" 이렇게 여러개 입력하면 api 2번 호출해서 모든 데이터 반환
- 검색조건 저장하면 다음에 같은 조건 불러와서 저장할 수 있게
