# Project Summary
The Bidding Notification System is a web application designed to simplify the search and management of bids, particularly for government contracts. It serves as a comprehensive platform that connects users with various government APIs, enabling efficient bid management, user authentication, and personalized preferences for a streamlined bidding experience.

# Project Module Description
The application consists of several functional modules, including:

- **Bid Search**: Users can search for bids using various parameters and save their search preferences.
- **Bid Calculator**: Tools for calculating potential bid amounts based on user-defined criteria.
- **User Authentication**: Secure registration and login processes using JWT tokens.
- **User Preferences**: Management of user-specific search conditions and saved searches.
- **User Interface Components**: Developed with Shadcn-UI for a cohesive, responsive design.

# File Description Inventory
- **api.ts**: Functions for interacting with the bidding API, including bid search and document download functionalities.
- **BidCalculator.tsx**: Component for calculating bid amounts.
- **BidTable.tsx**: Component for displaying bid information.
- **SearchForm.tsx**: Component for submitting search queries.
- **ui/**: Contains various reusable UI components such as buttons, modals, and forms.

# Technology Stack
- **Frontend**: React, TypeScript, Shadcn-UI, Tailwind CSS
- **Backend**: FastAPI, PostgreSQL, SQLAlchemy (async), JWT for authentication, HTTPX for API calls

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all necessary packages.
2. **Build the Project**: Run `pnpm run build` to compile the application.
3. **Run the Application**: Use `uvicorn app.main:app --reload --host 0.0.0.0 --port 8000` for the backend and `pnpm run dev` for the frontend.
